Download後のinstallationについては，[Install](../Install)のページを参照してください．

### Source code (GitHub)
Source codeは，[GitHub](https://github.com/CueMol/cuemol2/)からdownloadできます．
```
git clone https://github.com/CueMol/cuemol2.git cuemol2
```
